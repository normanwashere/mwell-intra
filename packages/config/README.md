# @intra/config

Shared build/lint/style presets for the Mwell Intra monorepo.

- `tsconfig/base.json` — base TS options (extends root `tsconfig.base.json`)
- `tsconfig/react-library.json` — for shared React packages
- `tsconfig/nextjs.json` — for the Next.js shell + modules
- `eslint/base.mjs` — shared flat ESLint config
- `tailwind/preset.cjs` — semantic-token Tailwind preset (definitions in `@intra/ui`)
